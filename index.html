Facebook Car Bot

When I started used car shopping I found how tedious it is to search through Facebook marketplace to find a good deal. Instead of struggling with it, I used Python to automate the process. Using a spreadsheet filled with cars, my Python bot would give each car a score based on year, milage, and price of the car. Then, it would search through hundreds of car listings and try to find cars with a higher score. When a listing had a higher score, it would be added to the spreadsheet for me to check-on later.

Discord Bot

A club at BSM started using Discord as their communication platform when the school was shutdown. With many people in the server it was hard for the leaders to keep up with everything. To fix the problem, I used the Discordpy library to make a bot. The bot's main function was to handle new user information and assign roles but could also be used for server moderation.
